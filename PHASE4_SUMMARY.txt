┌─────────────────────────────────────────────────────────────────────────────┐
│ ✨ CARACTERÍSTICAS IMPLEMENTADAS
└─────────────────────────────────────────────────────────────────────────────┘

    GRÁFICOS OHLC:
    ✅ Candlestick verde (UP) y rojo (DOWN)
    ✅ Auto-escalado de ejes
    ✅ Grid personalizado
    ✅ Zoom interactivo
    ✅ Pan automático

    INDICADORES SUPERPUESTOS:
    ✅ RSI(14)           → Línea naranja, niveles 30/70
    ✅ MACD(12,26,9)    → Cyan + Signal Magenta + Histograma
    ✅ Bollinger(20,2)  → Bandas superior/media/inferior
    ✅ Liquidaciones    → Histograma verde/rojo

    CÁLCULOS PRECISOS:
    ✅ RSI: Fórmula RS = avgGain/avgLoss
    ✅ MACD: EMA rápido/lento + Signal line
    ✅ Bollinger: SMA + Desviación estándar
    ✅ EMA: Factor exponencial k = 2/(n+1)

    SEÑALES AUTOMÁTICAS:
    ✅ RSI:       OVERSOLD (<30) → BUY, OVERBOUGHT (>70) → SELL
    ✅ MACD:      BULLISH → BUY, BEARISH → SELL
    ✅ Bollinger: LOWER_BAND → BUY, UPPER_BAND → SELL
    ✅ Overall:   STRONG_BUY, STRONG_SELL, NEUTRAL

    UTILIDADES:
    ✅ Fetch histórico desde APIs
    ✅ Validación automática de datos
    ✅ Normalización multi-fuente
    ✅ Merge inteligente sin duplicados
    ✅ Caché localStorage con expiración
    ✅ Export a CSV
    ✅ Manejo de errores


┌─────────────────────────────────────────────────────────────────────────────┐
│ 🚀 INICIO RÁPIDO (3 pasos)
└─────────────────────────────────────────────────────────────────────────────┘

    PASO 1: Copiar archivo
    $ cp index_with_charts.html index.html

    PASO 2: Abrir en navegador
    (Tu servidor local con index.html)

    PASO 3: Hacer clic en "Cargar Datos Demo"
    ¡Listo! Gráfico con indicadores funcionando

    ALTERNATIVA: Integración manual
    <script src="lib/chart-renderer.js"></script>
    <script src="lib/indicators-visual.js"></script>
    <div id="chart" style="height: 500px;"></div>
    <script>
      const chart = new ChartRenderer('chart');
      const data = await ChartDataHelper.fetchHistoricalData('BTC', '1h', 100);
      chart.loadPriceData(data);
      const indicators = IndicatorsVisual.calculateAllIndicators(data);
      chart.addRSI(indicators.rsi);
      chart.addMACD(indicators.macd);
    </script>

    OHLC Format:
    {
      time: 1676000000,    // Unix timestamp (segundos)
      open: 67000,
      high: 67500,
      low: 66800,
      close: 67200,
      volume: 1000000
    }

    RSI Output:
    { index: 14, value: 65.4 }  // 0-100

    MACD Output:
    { macd: -0.0245, signal: -0.0198, histogram: -0.0047 }

    Signals:
    {
      rsiSignal: { type: 'OVERSOLD', value: 28.5, action: 'BUY' },
      macdSignal: { type: 'BULLISH_CROSSOVER', action: 'BUY' },
      bbSignal: { type: 'AT_LOWER_BAND', action: 'BUY' },
      overall: 'STRONG_BUY'
    }

│ 🔧 TROUBLESHOOTING

    "Gráfico no aparece"

    → Verificar ohlcData.length >= 35
    → Usar ChartDataHelper.validateOHLCData()

    "APIs no responden"
    → Revisar endpoints en ChartDataHelper
    → Usar demo data como fallback
