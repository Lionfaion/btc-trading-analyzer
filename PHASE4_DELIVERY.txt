  ✅ MACD(12,26,9) → Cyan + Signal Magenta + Histograma
  ✅ Bollinger Bands(20,2) → Bandas semi-transparentes + SMA
  ✅ Liquidation Heatmap → Histograma verde/rojo

CÁLCULOS:
  ✅ RSI: Fórmula precisa con RS = avgGain/avgLoss
  ✅ MACD: EMA rápido/lento + Signal EMA
  ✅ Bollinger: SMA + Desviación estándar
  ✅ EMA: Factor exponencial k = 2/(n+1)

SEÑALES DE TRADING:
  ✅ RSI Signal: OVERSOLD (<30) → BUY, OVERBOUGHT (>70) → SELL
  ✅ MACD Signal: BULLISH_CROSSOVER → BUY, BEARISH → SELL
  ✅ Bollinger Signal: AT_LOWER_BAND → BUY, AT_UPPER → SELL
  ✅ Overall Signal: Mayoría de votos (STRONG_BUY/SELL/NEUTRAL)

ESPECIFICACIONES TÉCNICAS
================================================================================

TECNOLOGÍA:
  - Frontend: HTML5 + JavaScript (ES6+)
  - Gráficos: TradingView Lightweight Charts v4.1.0
  - Formato: Módulos reutilizables (window global)
  - Validación: Automática en entrada de datos
  - Caché: LocalStorage con timestamp

COMPATIBILIDAD:
  - Navegadores: Chrome, Firefox, Safari, Edge (últimas 2 versiones)
  - Resolución: Responsive a partir de 768px
  - Performance: Soporta hasta 200 candles sin lag

REQUISITOS DE ENTRADA:
  - OHLC Data: mínimo 35 candles para MACD
  - Timestamp: Unix en segundos (no milisegundos)
  - Precios: Flotantes válidos
  - Validación: Automática, high >= low

SALIDA DE DATOS:
  - Indicadores: Arrays con { time/index, value }
  - Señales: { type, value/action }
  - Overall: 'STRONG_BUY' | 'STRONG_SELL' | 'NEUTRAL'

USO RÁPIDO
================================================================================

1. INICIALIZAR GRÁFICO:
   const chart = new ChartRenderer('chart');

2. CARGAR DATOS:
   const ohlcData = await ChartDataHelper.fetchHistoricalData('BTC', '1h', 100);
   chart.loadPriceData(ohlcData);

3. CALCULAR INDICADORES:
   const indicators = IndicatorsVisual.calculateAllIndicators(ohlcData);

4. MOSTRAR EN GRÁFICO:
   chart.addRSI(indicators.rsi, '#ff9900');
   chart.addMACD(indicators.macd);
   chart.addBollingerBands(indicators.bollingerBands);

5. GENERAR SEÑALES:
   const signals = IndicatorsVisual.getSignals(indicators, currentPrice);
   console.log(signals.overall); // 'STRONG_BUY', 'STRONG_SELL', 'NEUTRAL'
